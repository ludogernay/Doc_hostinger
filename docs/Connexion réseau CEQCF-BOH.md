# 🌐 Connexion au réseau CEQCF-BOH

## 📝 Objectif
Ce document détaille les étapes pour se connecter à un réseau caché CEQCF-BOH sur **Windows, macOS, Android et iOS**.

---

## 📌 Prérequis
- Avoir les **informations du réseau** :
  - **Nom du réseau (SSID)** : CEQCF-BOH
  - **Type de sécurité** : WPA2-Personnel
  - **Mot de passe** : 41F&stJ86eqK&r
- Être dans une **zone de couverture du Wi-Fi**

---

## 🖥️ **Connexion sous Windows**
1. Cliquez sur l’**icône réseau** dans la barre des tâches (en bas à droite).
2. Sélectionnez **Wi-Fi**, puis cliquez sur **Autres paramètres Wi-Fi**.
3. Cliquez sur **Gérer les réseaux connus**.
4. Sélectionnez **Ajouter un réseau**.
5. Dans la boîte de dialogue qui s’ouvre :
   - **Nom du réseau** : Saisissez le SSID du réseau caché.
   - **Sécurité** : Sélectionnez **WPA2-Personnel**.
   - **Mot de passe** : Entrez le mot de passe.
   - **Cochez** :
     - ✅ **Se connecter automatiquement**
     - ✅ **Se connecter même s’il ne s’agit pas d’un réseau de diffusion**
6. Cliquez sur **Enregistrer** ou **Connecter**.

---

## 🍏 **Connexion sous macOS**
1. Cliquez sur l’**icône Wi-Fi** dans la barre de menus en haut à droite.
2. Sélectionnez **Autres réseaux** > **Autre…**.
3. Dans la fenêtre qui s’ouvre :
   - **Nom du réseau** : Saisissez le SSID.
   - **Sécurité** : Sélectionnez **WPA2-Personnel**.
   - **Mot de passe** : Saisissez le mot de passe.
4. Cliquez sur **Se connecter**.

---

## 🤖 **Connexion sous Android**
1. Allez dans **Paramètres**.
2. Sélectionnez **Wi-Fi** ou **Réseaux & Internet**.
3. Activez le Wi-Fi s’il est désactivé.
4. Cliquez sur **Ajouter un réseau** ou l’icône **+**.
5. Remplissez les champs :
   - **Nom du réseau (SSID)**
   - **Sécurité** : Sélectionnez **WPA2-Personnel**.
   - **Mot de passe** : Saisissez le mot de passe.
6. Cliquez sur **Enregistrer** ou **Connecter**.

---

## 📱 **Connexion sous iOS (iPhone)**
1. Allez dans **Réglages** > **Wi-Fi**.
2. Activez le Wi-Fi.
3. Cliquez sur **Autre...** en bas de la liste des réseaux disponibles.
4. Remplissez les champs :
   - **Nom du réseau (SSID)**
   - **Sécurité** : Sélectionnez **WPA2-Personnel**.
   - **Mot de passe** : Entrez le mot de passe.
5. Cliquez sur **Rejoindre**.

---

## ❗ Remarques et dépannage
- **Si la connexion ne fonctionne pas** :
  - Vérifiez les **informations du réseau**.
  - Assurez-vous d’être **dans la zone de couverture**.
  - Redémarrez le **Wi-Fi** de votre appareil.
- En cas de problème persistant, **contacter le service informatique** : si@canopycannes.com ou **5021**.
