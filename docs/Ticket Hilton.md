# 📝 Ouverture d'un Ticket d'Assistance Hilton (ServiceNow)

## 🎯 Objectif
Décrire le processus de création d'une demande d'assistance sur **ServiceNow** pour signaler un problème de connexion **RDP (Remote Desktop Protocol)** au serveur Hilton.

---

## 🛠️ Étapes détaillées

### 1️⃣ **Accéder à ServiceNow**
1. **Ouvrir un navigateur web**.
2. Aller à l’URL : [https://hilton.service-now.com](https://hilton.service-now.com).

---

### 2️⃣ **Se connecter**
1. **Entrer vos identifiants Hilton** (*nom d'utilisateur et mot de passe*).
2. **Cliquer sur Connexion**.

---

### 3️⃣ **Créer une demande d'assistance**
1. **Naviguer** vers la section **Report an Issue**.
2. **Remplir le formulaire** :
   - **Issues Details** : Sélectionner **Hardware**.
   - **Type de problème** : Sélectionner **Something else (enter details below)**.
   - **Description du problème** : Expliquer l'erreur rencontrée, exemple :
     ```
     Unable to connect to the server via RDP. The affected server IP is 10.149.482.27. The error message says "Remote Desktop connection failed". I have restarted my computer and checked my internet connection, but the problem persists.
     ```
   - **Numéro de série du matériel** : `N/A`.
   - **Emplacement du matériel** :
     ```
     Server Room, 2nd Floor, Canopy by Hilton Cannes, 2BD Jean Hibert, 06400 CANNES
     ```

---

### 4️⃣ **Ajouter des pièces jointes**
- **Captures d’écran des erreurs** (*si disponibles*).
- **Documents ou logs pertinents**.

---

### 5️⃣ **Soumettre la demande**
1. **Vérifier les informations**.
2. Cliquer sur **Submit** pour envoyer la demande.

---

### 6️⃣ **Suivi du ticket**
- Un **numéro de ticket** est généré après soumission.
- Utiliser ce numéro pour **suivre la demande** via ServiceNow.
- Contacter le support Hilton si nécessaire.

---

## ❗ Conseils supplémentaires
- **Sélectionner le bon niveau d’urgence** si le problème impacte les opérations de l'hôtel.
- **Vérifier la connexion internet et redémarrer la machine** avant de soumettre le ticket.
- **En cas de doute, contacter le service informatique** : si@canopycannes.com ou **5021**.
