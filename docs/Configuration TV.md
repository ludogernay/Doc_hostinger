# 📺 Installation et Configuration des TV

## 📝 Objectif
Ce document décrit les étapes pour configurer et installer les télévisions dans les chambres de l'hôtel.

---

## 📌 Prérequis
- **TV connectée aux switch Guest Tek**
- **Télécommande** de la télévision
- **Code d'accès** *(à demander au préalable)*

---

## 🛠️ Étapes détaillées

### 1️⃣ **Paramètres initiaux à la mise sous tension**
1. **Allumer la télévision**.
2. Sélectionner la **région locale** : `EU Spain`.
3. Choisir **Basic Setup**.
4. Sélectionner l’**emplacement** : `France`.
5. **Configurer un code PIN** *(demander le code si nécessaire)*.
6. **Accepter les conditions générales**.
7. Passer la création de compte.
8. Passer l’inscription au programme **Samsung Enterprise**.
9. Dans le menu autonome **Samsung** : Régler `Smart Service - Apps Editable` sur `On`.
10. **Éteindre la TV**.

---

### 2️⃣ **Paramètres généraux dans le menu principal Samsung**
1. **Allumer la TV**.
2. Désactiver le **retour sonore** :
   - `Menu → Son → Paramètres experts → Retour sonore : OFF`
3. Désactiver **Eco-solution** :
   - `Menu → Général → Eco Solution → Désactiver toutes les options`
4. Désactiver **l’économiseur d’écran** :
   - `Menu → Général → Gestionnaire de système → Temps de protection automatique : OFF`
5. Désactiver **Anynet+** :
   - `Menu → Général → Gestionnaire de périphérique externe → Anynet+ (HDMI-CEC) : OFF`

---

### 3️⃣ **Paramètres dans le menu d’accueil Samsung**
1. **Accéder au menu d'accueil** :
   - Appuyer sur le **bouton d'accueil** de la télécommande.
2. **Désactiver Samsung TV Plus** :
   - Sélectionner **Samsung TV Plus** → Appuyer sur la **flèche vers le bas** → **Supprimer** → **Confirmer la suppression**.

---

### 4️⃣ **Paramètres dans le menu Hôtel**
1. **Accéder au menu autonome** :
   - Suivre le schéma d’accès (*Code à demander au préalable*).
2. Désactiver l'affichage du **Menu OSD** :
   - `Menu OSD → Menu Display: OFF`
3. Configurer les **services intelligents** :
   - `Smart Services → Apps Editable: OFF`
4. Activer le **mode H.Browser** :
   - `H.Browser Solution → H.Browser Mode: ON`
5. Configurer l'**URL du lanceur** :
   - `https://setupmy.tv` (ou `https://setupmy.tv/t6` pour Tizen 6++)
6. Activer le **mode de veille virtuel** :
   - `Virtual Standby → Virtual Standby Mode: ON`

---

### 5️⃣ **Finalisation et redémarrage**
1. **Éteindre et rallumer la télévision**.
2. **Entrer le code d’accès** *(à demander au préalable)*.
3. Renseigner le **numéro de chambre** *(optionnel)*.
4. Vérifier que l'installation est complète si **le rooftop apparaît**.

---
