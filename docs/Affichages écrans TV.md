# 🖥️ Gestion des affichages sur écrans (Meeting Rooms & Chambres)

## 📝 Objectif
Décrire les étapes pour gérer l’affichage dynamique sur les **écrans des salles de réunion**, **ascenseurs**, **réception**, et **TV en chambre** via le **Cloud Portal**.

---

## 🔐 Connexion au portail
- **Lien** : *https://thecloudportal.com/*
- **Identifiant** : adresse email
- **Mot de passe** : Mot de passe défini

---

## 📺 Affichage dynamique – Salles de réunion / Ascenseurs / Réception

### 1️⃣ **Ajout de contenu**
- Aller dans : `Contenu → Média`
  - Pour les **écrans salons** : choisir **Affichage dynamique**.
  - Pour les **écrans réception/ascenseurs** : choisir **Signage** puis **Vidéo**.

### 2️⃣ **Planification et affichage**
- Aller dans : `Affichage → Planification`
- Sélectionner l’écran concerné → onglet **par défaut**
- Glisser/supprimer les contenus souhaités.
- **Sauvegarder** puis **mettre à jour (update)**.

🔥 **Astuce** : planifier à l’avance dans `Affichage → Planification → Calendrier`, cliquer sur la date/heure, puis faire glisser le contenu à afficher.

---

## 🛏️ Affichages TV en chambre

### 🎨 **Changer l’image de fond**
1. Aller dans : `Contenu → Média → Signage Default Content → Image`
2. Aller dans : `Modèle de Télévision`, double-cliquer sur l’image de fond.
3. Accéder à `Default Parameters`, cliquer sur l’icône ⚙️, puis **Image IRL**.
4. Cliquer sur l’image à ajouter → **Copier l’adresse URL affichée dans Value**.
5. Aller dans l’onglet `CSS`, remplacer l’ancienne URL par la nouvelle.
6. Cliquer sur **Apply and Close**.

⚠️ **Prévoir l’affichage la veille** : la mise à jour peut être différée sur certains écrans.

---

### 🍽️ **Modifier les encarts de menu**
1. Double-cliquer sur le **rectangle** à modifier.
2. Dans `Normal Content`, changer le **nom**.
   - ⚠ **Changer aussi dans les autres langues** si le nom est modifié.

### 🖼️ **Changer le contenu**
- Comme pour l’image de fond : récupérer l’URL de la nouvelle image et la remplacer.

### ➕ **Ajouter un nouvel encart**
- Copier/coller un encart existant.
- Modifier son **nom** et **contenu**.

### 🎮 **Modifier la navigation à la télécommande**
- Aller dans : `Eventhandlers`
- Définir les transitions entre encarts (par ex. : Room Service ➡ Restaurant Info ➡ Spa & Fitness ➡ Hotel Info ➡ Green).
